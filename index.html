About

Slim text is a slim text editor which runs inside a web browser. Currently it only supports Google Chrome and it is cross-platform (Windows, Linux and Mac OS X).

Installation

You can install it from Chrome Web Store. And don't forget to give it a 5 star rating ^_^

Alternatively, you can install it manually:

  1. Download newest.crx (version 0.0.8)
  2. Open the Chrome extentions page:
    1. Click the Chrome menu on the browser toolbar
    2. select Tools
    3. select Extensions
  3. Drag and drop newest.crx onto Chrome extensions page
